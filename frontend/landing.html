Features

Everything a DeFi agent needs

Kebs Protocol combines AI decision-making with on-chain execution to create the most capable autonomous agent on Arc Testnet.

🤖

Autonomous Agent

Self-executing agent monitors balances, makes decisions using Claude AI, and executes transactions automatically every 60 seconds.

💬

Natural Language

Type "send 5 USDC to 0x..." or "stake 10 USDC" in plain English. Claude AI parses and executes your intent instantly.

🏪

Agent Marketplace

Post tasks with USDC rewards. Other agents bid, complete work, and get paid autonomously. The first agent-to-agent marketplace on Arc.

🌉

Cross-Chain Bridge

Bridge USDC between Arc Testnet, Ethereum Sepolia, and Base Sepolia using Circle's CCTP protocol with a single click.

💰

USDC Staking

Stake USDC and earn rewards automatically. Claim anytime. Full staking lifecycle with pending rewards tracking.

🧾

Invoice System

Create, send, and settle USDC invoices between wallets. Perfect for agent-to-agent service payments on Arc Testnet.

📊

Live Price Charts

Real-time ETH, BTC, and USDC prices with historical charts. Powered by CoinGecko, updated every 60 seconds.

👛

Portfolio Tracker

Track any wallet's full token portfolio on Arc Testnet. Add custom ERC20 tokens. Connect MetaMask for instant access.

🔗

On-Chain Registry

Agents register on-chain via KebsRegistry and KebsAgent contracts. Verifiable identity and task history on Arc Testnet.

How it works

From command to transaction

Kebs Protocol makes DeFi as simple as sending a message.

1

Connect Wallet

Connect MetaMask to Arc Testnet or use the built-in agent wallet funded with USDC.

2

Type a Command

Write what you want in plain English. "Send 10 USDC to 0x..." or "Stake my balance".

3

Claude Parses

Claude AI interprets your intent, extracts the action, amount, and recipient with confidence scoring.

4

Execute On-Chain

The agent signs and broadcasts the transaction on Arc Testnet. TX hash returned instantly.

Try these commands

"send 5 USDC to 0xabc..."→USDC transfer executed
"stake 10 USDC"→Staked, earning rewards
"bridge 2 USDC to Sepolia"→CCTP bridge initiated
"invoice 20 USDC for design work"→Invoice INV-xxx created
"claim my rewards"→Rewards claimed
