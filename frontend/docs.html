KEBS PROTOCOL

API v1.0 Arc Testnet Dashboard

Overview

Base URL: https://kebsprotocol.onrender.com

All endpoints return JSON. No authentication required for read endpoints.
Agent Portfolio Staking Bridge Invoices Prices Protocol Chat

Agent

GET/agent/wallet
Get agent wallet address and balances
GET/agent/status
Get agent running status and recent logs
POST/agent/start
Start the autonomous agent loop
Body: { "interval": 60000 }
POST/agent/stop
Stop the autonomous agent loop
GET/agent/monitor
Get monitored addresses and on-chain events

Portfolio

GET/portfolio/:address
Get full token portfolio for any wallet address
POST/portfolio/token
Add a custom ERC20 token to track
Body: { "address": "0x...", "tokenAddress": "0x..." }

Staking

GET/staking/stats
Get global staking statistics
POST/staking/stake
Stake USDC to earn rewards
Body: { "address": "0x...", "amount": 10 }
POST/staking/unstake
Unstake USDC and collect rewards
Body: { "address": "0x...", "amount": 10 }
POST/staking/claim
Claim pending staking rewards
Body: { "address": "0x..." }

Bridge

GET/bridge/chains
Get list of supported chains
POST/bridge/transfer
Bridge USDC between chains via Circle CCTP
Body: { "fromChain": "arc", "toChain": "sepolia", "amount": 1, "recipient": "0x..." }

Invoices

GET/invoices
Get all invoices. Filter: ?address=0x...
GET/invoices/stats
Get invoice statistics
POST/invoices/create
Create a new USDC invoice
Body: { "from": "0x...", "to": "0x...", "amount": 10, "description": "Service fee", "dueDate": "2026-06-01" }
POST/invoices/pay/:id
Mark invoice as paid
Body: { "txHash": "0x..." }

Prices

GET/prices/latest
Get latest prices for USDC, ETH, BTC
GET/prices/history/:coin
Get price history (usdc, eth, btc)

Protocol

GET/protocol/info
Get protocol info and capabilities
GET/protocol/agents
Discover all registered agents

AI Chat

POST/chat/message
Send a message to the Kebs Protocol Claude AI
Body: { "message": "What is my agent balance?" }
GET/chat/history
Get chat history
